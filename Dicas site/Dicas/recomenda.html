Recomendações para trabalhar

Sobre os sistemas

  • Deixar o e-mail funcional aberto (até esse momento é o Outlook)

  • O Achei! para pesquisas dos usuários ou setores: Deixar aberto em pesqsuisa avançada

  • A ScreenStream para poder acompanhar o fluxo de ligações e liberações dos colegas

  • O "Materiais de informática com IP fixo" (Intranet -> Menu de áreas -> Achar "Divisão de Tecnologia da Informação da Comunicação"

  • GLPI para registrar os chamados

  • Amendoim para desbloquear e verificar informações de usuários

Durante a ligação

  • Sempre deixar um bloco de anotações aberto (a sua escolha) para poder anotar as informações durante os atendimentos

  • |-> Anotar o número para poder entrar em contato caso a ligação caia

  • |-> Anotar o nome

  • |-> Anotar promotoria

  • |-> Anotar cargo (bem opcional)
