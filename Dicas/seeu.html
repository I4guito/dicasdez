Informações Importantes

Em caso de dúvidas sobre orientar o usuário, passar para o sistemas

Podem ser enviadas minutas .doc, porém versão definitiva deve ser .pdf

O encaminhamento de um processo para um agente administrativo deve ser feito pelo próprio agente (caso seja para outro)


Logar pela primeira vez depois de ser cadastrado corretamente

  • |-> Login e senha são fornecidos por e-mail cadastrado, então se estiver sido colocado errado, pedir para quem o cadastrou arrumar isso

  • |-> Login: é sempre o cpf preferencialmente, sendo que no final pode ter alguma coisa extra
    \_> Estagiários não tem esse final extra

  • |-> Senha: a primeira é fornecida via e-mail cadastrado, as demais são trocados dentro do sistema

Quem cadastra quem

  • |-> Gerente da promotoria cadastra promotor.

  • |-> Promotor cadastra assessor e estagiário.

  • |-> GERENTE NÃO CADASTRA GERENTE. É uma lista especial. Caso o gerente irá entrar de férias, ele deverá falar com a UGP (Gestão de Processos - 8331)

    • Caso precise cadastrar um promotor e este seja de POA, passar para a UGP

  • |-> Não temos como saber corretamente quem é o gerente responsável por cadastrar

    • Promotor precisa ser habilitado na promotoria para acessar os processos

  • |-> Para habilitar será necessário que o Gerente faça isso na Habilitação

    • Visualizar vídeos:

  • |-> Abrir com o Firefox e ver se consegue executar os vídeos

    • Não consegue assinar

  • |-> SE FOR BANRISUL: Verificar se o certificado do Banrisul está instalado
    \_> Perguntar se não é o cartão que contém a conta pessoal também, pois o SEEU dá problema se os dois estiverem no mesmo (certificado digital e conta pessoal)

  • |-> SE FOR VALID: Verificar se o certificado do Valid está instalado

    • Erro ao assinar de forma consecutiva:

  • |-> Erro que ocorre é o: "O Cartão Inteligente não pode executar a operação solicitada"

  • |-> Tem que estar no Mozilla Firefox

  • |-> Ir nas "Opções" do Firefox

  • |-> Ir em "Avançado"

  • |-> Ir em "Certificados"

  • |-> Clicar em "Ver certificados"

  • |-> Adicionar a DLL do Banrisul
    \_> Ir em "C:"
    \_> Clicar em "Arquivos de programa (x86)"
    \_> Clicar em "CSP Banrisul Múltiplo"
    \_> Selecionar a DLL ali presente

  • |-> Adicionar a DLL do Valid
    \_> CertiSign/Valid
    32 bits
    C:\Windows\SysWOW64\aetpkss1.dll

    • \_> CertiSign/Valid
    64 bits
    C:\Windows\System32\aetpkss1.dll
  • |-> Limpar o cache e tentar de novo

    • Desabilitar promotor(a) que está habilitado em outra promotoria

  • |-> Somente o gerente de cada promotoria pode habilitar/desabilitar o promotor vinculado, sendo assim somente através de requisição para outra promotoria pedindo para desabilitar

    • Quando há mais de 1 perfil

  • |-> Para acessar como Gerente precisa digitar: cpf.germp

  • |-> Se não conseguir acessar outro perfil, possível problema de senha diferente entre perfis
    \_> Quando a pessoa vai trocar a senha, há um campo de criar uma senha diferente para cada perfil

    • Caso o CPF tenha sido cadastrado errado:

  • |-> Enviar e-mail para a ugp@mprs.mp.br

  • |-> Informar que é sobre o SEEU

  • |-> Colocar o login errado (o CPF) e o CPF correto no e-mail
