Leitor de Cartão

Em caso de problema, pedir para solicitarem outro (se não for possível arrumar)

  • Externo: PertoSmart
    \_> Em geral, só instalar o driver (\Drivers\Leitoras de cartão\Perto EMV Windows 8) que a leitora volta a funcionar

  • Do teclado:Se for DELL
    \_> FAZER ISSO BOA PARTE DAS VEZES: usar o driver genérico da Microsoft
    \_> Ir no "Gerenciar dispositivos"
    \_> "Leitor de cartão"
    \_> Desinstalar o Driver se for outro
    \_> O sistema irá reinstalar automaticamente, então pedir para o usuário testar se funciona
    \_> Ver se o sistema detecta o leitor de cartão
    \_> Instalar o driver genérico (\\uauserver2\KitApoio\Drivers\Teclados SmartCard\Dell_smartcard_generico)
    \_> Instalar o gkuprod2d.msi da versão do computador
  • Se for HP:
    \_> Não pode ter o cartão conectado
    \_> Instalar o driver (\\uauserver2\KitApoio\Drivers\Teclados SmartCard\Teclado HP6305 v2.3)
    \_> Seguir o manual e testar com o usuário
