Comandos úteis para usar no cmd:

--------------------------------------------------------------------------------------------------------------------------------

Testar a conectividade (pingar)

ping t??????

no caso seria ping + tombo (ping t123456)

--------------------------------------------------------------------------------------------------------------------------------

Ficar pingando continuamente

ping t?????? -t


--------------------------------------------------------------------------------------------------------------------------------

saber as permissões do usuário

gpresult /r


--------------------------------------------------------------------------------------------------------------------------------

forçar atualizações de política de rede no computador

gpupdate /force


--------------------------------------------------------------------------------------------------------------------------------

reiniciar o computador a distância

shutdown -r -m \\tombo -f -t 0


--------------------------------------------------------------------------------------------------------------------------------

Verificar a integridade de todos os arquivos do sistema protegidos e reparar os arquivos com problemas quando possível

sfc /SCANNOW


--------------------------------------------------------------------------------------------------------------------------------

Saber se tem alguém conectado via VNC em alguma máquina:

psloggedon \t??????


--------------------------------------------------------------------------------------------------------------------------------

mostrar as principais informações úteis do seu computador

systeminfo


--------------------------------------------------------------------------------------------------------------------------------

mostrar as informações do endereço ip do computador

ipconfig


--------------------------------------------------------------------------------------------------------------------------------

limpeza do cache dns

ipconfig /flushdns


--------------------------------------------------------------------------------------------------------------------------------

faz análise de conexões tcp/ip ativos no computador

netstat


--------------------------------------------------------------------------------------------------------------------------------

mostra uma lista de todas conexões de rede abertas, a porta que está usando e o endereço ip externo conectada

netstat -an


--------------------------------------------------------------------------------------------------------------------------------

identifica todos os processos que estão em execução (naquele momento)

tasklist


--------------------------------------------------------------------------------------------------------------------------------

finaliza qualquer processo

taskkill /f /im nome_do_programa.exe


--------------------------------------------------------------------------------------------------------------------------------

lista todos os drivers instalados no computador
