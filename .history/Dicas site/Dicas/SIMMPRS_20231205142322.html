SIM MPRS (navegador portátil do Firefox versão 59.2)

Informações Importantes

  • -> É para ser instalado por política em todos os computadores, caso não esteja, só reiniciar ou no cmd digitar: gpupdate /force
  • -> Proxy já vem setado para pegar da máquina
  • -> Não salva cache e nem senhas, mesmo que queira, pois sempre entra como modo privativo
  • -> Serve para ter tudo para assinar e visualizar com o Flash, já que o mesmo foi bloqueado para sempre nos demais navegadores

Não exibe audiência porque aparecia um cone do VLC:

  • |-> Desabilitar o plugin / extensão do VLC no Firefox
  • |-> Desinstalar do computador o VLC e tentar acessar de novo a audiência

Se ocorre problema de "Falha ao carregar".

  • dar um CTRL + F5
  • Qualquer coisa desinstalar e reinstalar o "SIM MPRS"
  • |-> Pasta de cache do SIM MPRS: C:\Apps\SIM MPRS\FirefoxPortable\Data\profile\cache

Se ocorrer problema de Java:

  • Primeiro, se existir a pasta "C:\Windows\Sun\Deployment", apaga ela

Problema de java no SIM MPRS para assinar

  • |-> C:\Windows\Sun\Java\Deployment (tem que ver o caminho que dá quando ocorre o erro, peguei um que ficava C:\Users\usuario10\AppData\LocalLow\Sun\Java)
  • |-> Renomeia esta pasta (PREFERENCIALMENTE deletar a pasta) .old por exemplo
  • |-> Inserir novamente as exceções do Java
