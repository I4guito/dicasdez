O Word não abre

Nesse caso, são grandes as chances de que o modelo (Template) padrão esteja corrompido.

  1. Com o Word fechado clique no menu Iniciar, digite %appdata%\microsoft\modelos no campo de busca e tecle Enter.
  2. Você deverá ver uma janela do Windows Explorer com um ou mais arquivos, um deles chamado Normal, Normal.dotm ou Normal.dotx. Este é o modelo padrão.
  3. Se o usuário personalizou bastante o Word, pode tentar restaurar versões anteriores
  4. Mas se a janela diz Não há versões anteriores disponíveis, , a solução é forçar o Word a criar um novo modelo padrão.
  5. Clique com o botão direito do mouse sobre o arquivo Normal e escolha a opção Renomear.
