Como Criar um Novo Documento no Microsoft Word

Para criar um novo documento no Microsoft Word, siga os passos abaixo:

  1. Abra o Microsoft Word.
  2. Clique em "Arquivo".
  3. Selecione "Novo".
  4. Escolha o tipo de documento desejado.
  5. Comece a editar seu documento.
