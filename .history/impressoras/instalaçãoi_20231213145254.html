Instalar/configurar impressora

Passo 1: Pedir a tag ou tombo

Tag = Impressora locada/alugada
Tombo = Impressora pertence ao MP

Passo 2: Procurar IP usando Tecnologia da Informação e Comunicação (TIC - Filtro de busca)

Descreva o primeiro passo para resolver o problema. Forneça detalhes e, se possível, inclua uma imagem para ilustrar.


Disponível no link: https://intra.mp.rs.gov.br/site/accounts/login/?next=https://intra.mp.rs.gov.br/informatica/interno/materiais/index_html

Passo 3: Abrir "Gerenciamento de impressão"

Se quiser fazer pelo Painel de Controle, dá na mesma. Eu particularmente prefiro pelo gerenciamento de impressão

Passo 4: Ir até impressoras

Servidores de impressão >> Tombo local >> Impressoras

Passo 5: Adicionar uma impressora

Clique com o botão direito no espaço em branco >> Adicionar impressora...

Passo 6: Instalar utilizando IP

Se quiser procurar impressora na rede é mais fácil também


Após isso clique em Avançar >> Selecione em tipo de dispositivo a opção "TCP/IP" >> Cole o IP nas opções Nome de Host e Nome da porta


Passo 7: Se já estiver instalado, utilizar o driver existente para a impressora

Selecionar o Driver da impressora


Caso o Driver não exista:

Selecione instalar um novo driver >> Avançar >> Verifique se o Driver está listado


Se não tiver acesse: Instalar Driver

Passo 8: Após isso, Avançar, Avançar e Avançar

Depois disso, avançe a página até concluir a instalação. Em um desses "Avançar" terá uma opção para digitar como o nome da impressora irá aparecer para o usuário, caso não mexa, vai ficar com o nome do driver

Imprima uma página de teste após a conclusão e verifique o usuário se saiu.
Se imprimiu, OK! Se não imprimiu, verifique o motivo
