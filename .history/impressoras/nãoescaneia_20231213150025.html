Impressora não escaneia

|-> Impressora Samsung:

\_> Verificar se a porta está correta

\_> Verificar se o driver da impressora está selecionado corretamente no NAPS2

\_> Verificar se há outras impressora Samsungs (se tiver, tem que escolhar qual usar)

\_> Colar no twain_32 a seguinte opção: -- \\uauserver2\KitApoio\APLICATIVOS\NAPS2\CASO NÃO IDENTIFIQUE SCANNER\COLE EM C,WINDOWS,TWAIN_32

\_> Clicar para "Selecionar Scanner Samsung"

\_> Clicar para instalar por IP

\_> Colocar o IP do scanner e dar ok quando selecionar

\_> Testar se funciona (qualquer coisa utilize o vidro)

|-> Impressora HP:

\_> Só pode ter uma impressora HP configurada na máquina para funcionar

\_> Verificar se a porta está correta

\_> Verificar se o driver da impressora está selecionado corretamente no NAPS2

|-> Impressora Lexmark:

\_> Verificar se o driver da impressora está selecionado corretamente no NAPS2

\_> Verificar se a porta está correta

\_> Verificar se for a Lexmark MX431 (novas que substituíram as Samsungs)
