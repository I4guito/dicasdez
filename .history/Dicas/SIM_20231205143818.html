SIM

Informações Importantes

  • -> Sim, Judicial sempre é do tipo PJ. Quando for NF, DP, IC PA*, é extrajudicial
  • -> Ele tá programado para abrir na segunda tela as janelas que são abertas, porém só funciona no Firefox
  • -> SIM Judicial só aceita PDF!
  • -> Segundo o informado, a assinatura no SIM só funciona pelo aplicativo (firefox verde)

Limite máximo

  • |-> documentos: 50 MB
    \_> arquivos permitidos pelo SIM (que eu saiba)
    |-> .pdf
  • |-> vídeos: 50 MB
    \_> arquivos de vídeo permitidos pelo SIM (que eu saiba)
    |-> .mp4
    |-> .webm
  • |-> áudios: 50 MB
    \_> arquivos de áudio permitidos pelo SIM (que eu saiba)
    |-> .mp3
    \_> arquivos NÃO permitidos
    |-> .opus
    |-> .ogg
  • |-> imagens também são aceitas, mas não todas (preferencialmente enviar como um documento PDF ou inserir dentro de um Word e transformar em PDF depois):
    \_> .jpg e .png são aceitos
    \_> bitmap não é aceito

Não deixa juntar áudio e vídeo no SIM, só PDF

  • |-> Tem que ser feito no sistema diretamente (no caso o EPROC)

Editor do SIM

  • |-> Não tem como inserir nota de rodapé, pois a intenção é ser um editor simples de texto

Não consegue assinar

  • |-> Verificar se o java está executando (extensões/plugins)
  • |-> Verificar se o CSP Banrisul lê o certificado digital
    \_> Qualquer coisa precisa instalar o CSP Banrisul
    \_> Pode ser problema de driver
    \_> Pode ser problema de encaixe na porta usb
    \_> Pode ser problema de limpeza na parte do chipe do cartão
  • |-> Verificar se não há certificados antigos ou duplicados
  • |-> Verificar se o DLL tá inserido corretamente onde precisa
  • |-> Verificar se não está usando o SIM MPRS e ocorre erro de Java ao assinar ou tentar, aí pode consultar a sessão com nome "SIM MPRS"

Visualizar documento

  • |-> Para não dar nenhum problema, utilizar o Firefox
  • |-> Caso não abra alguma página do folheador (tipo Autos), verificar o Flash
    \_> Pressionar "Alt" do teclado ou clicar direito no navegador e abrir "Barra de Menus"
    \_> Ir em "Extensões"
    \_> Ir em "Plugins"
    \_> O Shockwave Flash deve estar em "Sempre ativar"
    \_> Dar Ctrl + F5 para ver se carrega a página da forma correta

Abre PDF fora do SIM (direto no Adobe Reader)

  • |-> Ir em opções
  • |-> Aplicativos
  • |-> Achar o "PDF" e colocar para "Visualizar no Firefox"
  • |-> Testar (ou recarregar a página e testar)

Página no folheador fica totalmente preta ou branca

  • |-> Dar Ctrl + F5
  • |-> Dar Ctrl + Shift + Del e limpar o cache
  • |-> Procurar opção de "Restaurar" para reconfigurar automaticamente o Firefox

Página no folheador fica com mensagem "Falha ao carregar"

  • |-> Se for PJ
    \_> Limpar cache e dar ctrl + f5
    \_> Verificar a versão do Flash
    \_> Usar o Google Chrome e ver se carrega
    \_> Falar com o avançado caso o problema persista

Sistema não carrega:

  • |-> Tentar acessar o SIM utilizando outros servidores:
    \_> http://sisprd1.mp.rs.gov.br:11180/sim/
    \_> http://sisprd3.mp.rs.gov.br:11180/sim/
    \_> http://sisprd4.mp.rs.gov.br:11180/sim/
    \_> http://sisprd4.mp.rs.gov.br:11280/sim/

Campo para digitar o PIN sumiu

  • |-> Verificar se há o plugin do Java no Firefox
  • |-> Se não há, verificar se o Java está instalado
    \_> Verificar se carrega a assinatura no Internet Explorer
    \_> Desinstalar o Java e reinstalar
    \_> Ver se funciona

    • Autos ou alguma outra coisa nunca termina de carregar

  • |-> Limpar cache do navegador (Ctrl + Shift + Delete)
  • |-> Caso ainda persista, usar o Ccleaner
  • |-> Pedir ajuda e possivelmente abrir chamado

Imprimir documento no folheador

  • |-> Só imprimi a página atual
  • |-> Para imprimir tudo será necessário transformar em PDF

    • Campos verdes no editor do SIM

  • |-> São TAGS, ou seja, o conteúdo é preenchido de outro ponto
  • |-> ou seja, não é possível editar parte do conteúdo, será necessário apagar e escrever o que quer depois no mesmo lugar

    • SIM Judicial

  • |-> Imprimir Despacho final/ Decisão final

Não carrega vídeos na juntada

  • |-> Na "Juntada" de vídeo não carrega a prévia de vídeos, tem que "Concluir" antes para poder ver

    • Problema de Flash (foi desabilitado pela empresa que o desenvolvia dia 12/01/21)

  • |-> Não há navegadores que consigam usar o Flash mais, somente a versão especial do Firefox chamado "SIM MPRS"
  • |-> É para estar instalado por política de rede em todos os computadores, caso não esteja, só reiniciar ou no cmd digitar: gpupdate /force

    • Não consegue logar no SIM ("Erro ao buscar grupos de trabalho")

  • |-> Verificar se ela foi autorizada a acessar o SIM (perguntar para o cadastro)
    \_> Se não, a chefia precisa enviar e-mail solicitando acesso
  • |-> Pedir para ela solicitar ao facilitador do SIM para ela ser vinculada a promotoria
  • |-> Às vezes pode ser falta de ter trocada a senha inicial
  • |-> Depois de ter sido colocado pelo facilitador e não conseguir entrar mesmo assim, trocar a senha pois pode ser um bug do SIM ainda existente

Rolagem nos marcadores do SIM:

  • |-> Tem marcadores demais, esse filtro funciona como um menu dropdown e não tem scroll
    \_> Talvez clicar em CTRL (sinal de menos>) resolva

Vídeos anexados no SIM só aparecem como um link no folheador

  • |-> Sim, isso é padrão, não tem o que fazer para o vídeo aparecer, só clicando nele mesmo para que possa ser visualizado

  • Valtoir Disse: o SIM Judicial só aceita arquivos multimídia em DILIGÊNCIAS Oriente o usuário a salvar a minuta da petição, pois será perdida. Na tela posterior à edição da minuta, tem a opção de retornar para intimação com prazo aberto. Selecionando essa opção, vai voltar pra tarefa "intimar-se", em que é possível registrar diligência No cumprimento da diligência (normalmente se registra tarefa de "Outras providências") é possível juntar arquivos de áudio/vídeo. E na tarefa de analisar o retorno, é possível peticionar e juntar os arquivos da DI.
